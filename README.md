# Shopping mall

Role: Back-end Dev, Front-end Dev, JPA, MySQL, Spring

소요시간: 2022.03 ~ 2022.03



# 주제

로그인, 상품 입력, 장바구니, 주문 등의 기본적인 쇼핑몰 기능을 구현함.

스프링부트 쇼핑몰 프로젝트 with JPA 책을 참고해 클론코딩을 진행함.
